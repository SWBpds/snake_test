# Snake Game

A classic Snake game built with Python and tkinter (no external dependencies).

## Controls

| Key | Action |
|-----|--------|
| Arrow keys / WASD | Move the snake |
| Space | Start / Restart |

## How to Run

```bash
python snake_game.py
```

## Features

- Snake grows and speeds up when eating food
- Score tracking with high score persistence
- Grid-based dark theme with neon visuals
- Game over on wall or self collision
